Project Description

EduPortal is a web-based School Management System developed using HTML, CSS, and JavaScript. The system provides an easy-to-use interface for managing school activities such as student records, teacher information, grades, attendance, and school notices. The project demonstrates the practical application of front-end web development technologies and user interface design.

Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JavaScript)

Main Features

User Authentication

- Login interface
- Role-based access control
- Administrator, Teacher, and Student roles

Student Management

- Add new students
- View student records
- Search students
- Remove students

Teacher Management

- Add teachers
- View teacher information
- Search teachers
- Remove teachers

Grades Management

- Record student grades
- Calculate grade categories
- Display academic performance

Attendance Management

- Mark attendance
- View attendance reports
- Generate attendance statistics

Notice Board

- Post announcements
- View school notices
- Manage notices

Dashboard

- School statistics overview
- Student and teacher summaries
- Attendance overview

Settings Module

- School information management
- System preferences
- Developer information
